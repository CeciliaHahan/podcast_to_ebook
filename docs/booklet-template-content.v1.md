- This file is content-only and should drive all format renderers from one source structure.

- For PDF: map each `chap_*` to visible section blocks with heading hierarchy.
- For Markdown: map each `chap_*` to `##/###` sections without changing order.
- `nav.xhtml` and `toc.ncx` should be generated from chapter titles after placeholders are resolved.
- Reject final output if any `{PLACEHOLDER}` remains.
